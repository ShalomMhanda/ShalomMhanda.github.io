Gliders Bike Sharing App

Project information

  • Category: Web App
  • Project timeline: February - May 2023

An intuitive and accessible web application facilitating local bike-sharing, designed for seamless user experience..

Key Features:

  • - Multi-language Support: Integrated with Google Translate API, making the app accessible to users in over 100 languages.
  • - Seamless Authentication: Effortless user login via Google or Facebook accounts.
  • - Smart Bike Reservation: Real-time bike availability and parking station tracking, powered by Google Maps, ensuring a smooth and convenient bike-sharing experience.
