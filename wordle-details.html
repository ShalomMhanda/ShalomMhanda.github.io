A mobile app that prompts a user to guess a random word.

Key Features:

  • - User can change the number of letters per row (the length of the goal word)
  • - User can change the numbers of rows on the board (how many guesses allowed)
  • - User can select a new themed set to pull the goal word from (theme options are Animals, Countries, and Random).
  • - User can select "alien wordle", causing the goal word to change after each guess (challenging).
  • - User can reset the game after playing to guess a new word.
